## User App Features for Ordering a Driver

This document outlines the key features and functionalities for the user-facing mobile application, focusing on the process of ordering a driver.

### 1. User Authentication & Profile Management

*   **Registration/Login:**
    *   Email/Password registration and login.
    *   Social media login (Google, Facebook).
    *   Forgot password functionality.
*   **Profile Management:**
    *   View and edit personal information (name, phone number, email).
    *   Add/manage payment methods (credit/debit cards, mobile wallets).
    *   View ride history.
    *   Manage saved addresses (home, work, etc.).
    *   Set preferred language.

### 2. Driver Ordering Process

*   **Location Services:**
    *   Automatic detection of current user location (GPS).
    *   Manual input of pickup location.
    *   Ability to set a different pickup location than current.
*   **Destination Input:**
    *   Search for destination by address, landmark, or point of interest.
    *   Autocomplete suggestions for addresses.
    *   Ability to add multiple stops (waypoints).
*   **Ride Type Selection:**
    *   Display available ride types (e.g., Standard, Premium, SUV, Motorcycle).
    *   Show estimated fare and arrival time for each ride type.
    *   Option to select number of passengers.
*   **Fare Estimation:**
    *   Real-time fare estimation based on distance, time, and ride type.
    *   Breakdown of fare components (base fare, per-mile, per-minute, surge pricing, tolls).
*   **Payment Method Selection:**
    *   Choose from saved payment methods.
    *   Option to add a new payment method during the booking process.
*   **Promo Codes/Discounts:**
    *   Apply promo codes before confirming the ride.
*   **Confirm Ride:**
    *   Review all ride details (pickup, destination, fare, payment).
    *   Confirmation button to request a driver.

### 3. During the Ride

*   **Driver Assignment & Tracking:**
    *   Display driver's name, photo, vehicle make/model, license plate number.
    *   Real-time tracking of driver's location on a map.
